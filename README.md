# VidyaVichara - Classroom Q&A Sticky Board

A real-time classroom Q&A platform where students can post questions during lectures as interactive sticky notes, and instructors can manage, filter, and respond to them efficiently.

## Features

### For Instructors

- Create and manage courses
- Start live Q&A sessions with unique session IDs
- View questions organized by students in a grid layout
- Filter questions by status (Unanswered, Answered, Important)
- Mark questions as answered or important
- Persistent storage for post-lecture review

### For Students

- Join sessions using session ID
- Post multiple questions during lectures
- View all questions from other students
- Real-time status updates when questions are answered
- Clean, intuitive interface for seamless interaction

## Tech Stack

- **Frontend**: React.js with responsive design
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **Real-time Communication**: Socket.io (planned)
